
# Calculator

A calculator app to make math operations.



## Tools

- React Hooks - useState
- props.children 
- CSS
- mathjs


## Resources & References

- [mathjs](https://mathjs.org/)

## How to Run?

To run this project on your system, run

```bash
  npm install
  npm start
```

To create a build of this project, run

```bash
  npm run build
```
